Matplotlib

Matplotlib is one of the most popular Python libraries for creating static, animated, and interactive visualizations in a variety of formats. It is particularly useful for data visualization in scientific and analytical fields. The core object of Matplotlib is its pyplot module, which provides a MATLAB-like interface for creating plots.

Matplotlib is often used with other libraries such as NumPy and Pandas to visualize data effectively. Some of the common types of plots you can create using Matplotlib include line plots, bar charts, histograms, scatter plots, pie charts, and more.

Basic Plotting

Matplotlibs pyplot interface allows you to quickly create basic plots like line plots, scatter plots, and histograms.
In[1]:
import matplotlib.pyplot as plt

# Basic Line Plot
x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y)
plt.xlabel('X Axis')
plt.ylabel('Y Axis')
plt.title('Basic Line Plot')
plt.show()

Figure and Axes

In Matplotlib, a figure is the overall window where plots are drawn, and an axes is an individual plot or graph inside the figure. A figure can contain multiple axes (or subplots).
In[2]:
fig, ax = plt.subplots()  # Creates a figure and a single axes
ax.plot(x, y)
ax.set_xlabel('X Axis')
ax.set_ylabel('Y Axis')
ax.set_title('Plot with Axes')
Out[2]:
Text(0.5, 1.0, 'Plot with Axes')

Subplots

You can create multiple plots (subplots) in a single figure:
In[6]:
fig, (ax1, ax2) = plt.subplots(1, 2)  # 1 row, 2 columns
ax1.plot(x, y)
ax1.set_title('Plot 1')

ax2.plot(y, x)
ax2.set_title('Plot 2')
plt.show()

Adding Labels, Titles, and Legends

Title: Add a title to the plot.

Legend: Add a legend to the plot to differentiate between multiple lines or data points.

Handling Dates

Matplotlib works seamlessly with date and time data using the dates module
In[24]:
import matplotlib.dates as mdates
import datetime as dt

dates = [dt.datetime(2024, 1, 1), dt.datetime(2024, 1, 2)]
values = [100, 200]

plt.plot(dates, values)
plt.gcf().autofmt_xdate()  # Auto formats the dates on x-axis

Types of Plots:

Line, Bar, Scatter, Histogram, Image, Box, Violin, Stream, Quiver, Area, Pie and Donut.
Import libraries, matplotlib.pyplot as plt.
x and y are input and output list.

Line Plot:

Basic line plot:
    plt.plot(x,y)
    plt.show()
    
Customize line plot:
    plt.plot(x, y, linewidth=2.0, linestyle=":", color="r", alpha=0.5, marker='o')
    plt.title('Title Name')
    plt.xlabel('xlabel name')
    plt.ylabel('ylabel name')
    plt.legend(['line1'], loc='best')
    plt.grid(True)
    plt.show()
    # linewidth = width of plotted line
    # linestyle = style of plotted line
    # color = color of plotted line
    # alpha = opacity of plotted line
    # marker = points in the plotted line

changing figsize :
    fig = plt.figure(figsize=(10,5))
    # not required to use fig in anywhere

adding 2 or more plots:
    x, y1, y2 are input and output variables.
    #first create the plot position in figure.
    plt.subplot(2,1,1)  # 2 rows, 1 column, 1 position
    plt.plot(x,y1)
    plt.title('Graph1')
    plt.subplot(2,1,1)  # 2 rows, 1 column, 2 position
    plt.plot(x,y2)
    plt.title('Graph2')
    plt.show()

Possible Interview Questions on Matplotlib

  1. What is Matplotlib, and why is it used?

     Matplotlib is a Python library for creating static, interactive, and animated visualizations. It provides an easy-to-use interface for plotting data and is widely used in data science for visualizing datasets, exploring data, and presenting results.

  2. What are the key objects in Matplotlib?

     The two key objects in Matplotlib are:
     Figure: The container for all plot elements.
     Axes: The area of the figure where the data is plotted (often called a subplot).

  3. How do you create a basic line plot in Matplotlib?

     Answer: You can use plt.plot() to create a basic line plot.

  4. How do you add a title, x-label, and y-label to a plot?

     plt.title('Title'): Adds a title to the plot.
 plt.xlabel('X Axis'): Labels the x-axis.
 plt.ylabel('Y Axis'): Labels the y-axis.

  5. What is the difference between plt.plot() and plt.scatter()?

     plt.plot() is used for line plots where points are connected by lines, while plt.scatter() is used to create scatter plots where individual data points are plotted without lines.

  6. How do you create multiple subplots in Matplotlib?

     You can use plt.subplots() to create multiple subplots in a figure.

  7. How do you enable grid lines in a plot?

     You can use plt.grid(True) to enable grid lines on the plot.

  8. How do you save a plot to a file in Matplotlib?

     You can save a plot using plt.savefig() to store the figure in a file format like PNG, PDF, or SVG.

  9. What are some common types of plots you can create using Matplotlib?

     Some common types of plots include line plots, bar plots, scatter plots, histograms, pie charts, and 3D plots.

  10. What is the use of plt.show() in Matplotlib?

    plt.show() is used to display the plot in a window or inline in a notebook. It renders the current figure and pauses the execution until the window is closed.

  11. How can you customize the appearance of a plot in Matplotlib?

    You can customize the appearance of plots by adjusting colors, line styles, markers, adding legends, and configuring axes using various arguments in the plotting functions (color, linestyle, marker, etc.).

  12. What is the purpose of fig, ax = plt.subplots() in Matplotlib?

    plt.subplots() is used to create a figure and one or more axes (subplots). This is useful for more complex plotting scenarios where you want to create multiple plots in a single figure.
