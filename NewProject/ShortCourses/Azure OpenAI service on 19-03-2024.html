Create an Azure OpenAI service:
	Subscribe azure first, then click on Azure OpenAI.
	Select subscription, Resource Group is the most important one, which store all the details of configuration.
	Region selection is next important one, it is due to model availability, cost, latency, network efficiency and performance.
	Name giving to the instance should be unique within Resource group, not required to unique within subscription.
	After giving basic details, go for network tab, we have to give permission through network. From which network can access it. The option are all access, 
		strict to separate internet, and private internet only.
	Final step to review and confirm, in this it will validate the access of you and policy you have to create it. If it is okay, it will create.
