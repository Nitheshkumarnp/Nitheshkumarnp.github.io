Homoscedasticity

Homoscedasticity is assessed by examining the residuals of a regression model, which are the differences between the observed values and the predicted values. Therefore, it can only be evaluated after fitting the model and obtaining the predictions.

What is Homoscedasticity?

    Homoscedasticity (or homoskedasticity) is a term used in statistics to describe a situation in which the variance of the errors (residuals) in a regression model is constant across all levels of the independent variables. In simpler terms, the spread or dispersion of the residuals remains consistent regardless of the values of the predictor variables.

Importance of Homoscedasticity

    Model Accuracy: Homoscedasticity is a key assumption in linear regression. If this assumption is violated (i.e., the residuals exhibit heteroscedasticity), the results of the regression analysis may be inefficient and biased.

    Inference Validity: Many statistical tests and confidence intervals rely on the assumption of homoscedasticity. Violating this assumption can lead to incorrect conclusions about the significance of predictors.

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.linear_model import LinearRegression

# Generate sample data
np.random.seed(0)
X = np.linspace(0, 100, 500)
y = 2 * X + np.random.normal(0, 10, size=X.shape)  # Homoscedastic errors

# Fit a linear regression model
X_reshaped = X.reshape(-1, 1)
model = LinearRegression()
model.fit(X_reshaped, y)
y_pred = model.predict(X_reshaped)

# Calculate residuals
residuals = y - y_pred

# Create a DataFrame for plotting
df = pd.DataFrame({'X': X, 'y': y, 'y_pred': y_pred, 'residuals': residuals})

# Plot the residuals
plt.figure(figsize=(12, 6))
sns.scatterplot(x='X', y='residuals', data=df)
plt.axhline(0, color='red', linestyle='--')
plt.title('Residual Plot')
plt.xlabel('Fitted Values (X)')
plt.ylabel('Residuals')
plt.show()

Interpretation of the Residual Plot:

    Random Scatter: The plot should show a random scatter of points around the horizontal axis (y=0). This indicates that the variance of the residuals is constant across the range of fitted values.

    No Pattern: There should be no discernible pattern (e.g., funnel shape, systematic increase or decrease) in the residuals. If there is any pattern, it would suggest heteroscedasticity.

Heteroscedasticity

What is Heteroscedasticity?

    Heteroscedasticity occurs when the variance of the errors (residuals) in a regression model is not constant across all levels of the independent variables. This means that the spread or dispersion of the residuals changes depending on the values of the predictor variables. In other words, the variability of the residuals is not uniform.

Why is Heteroscedasticity Important?

    Model Efficiency: Heteroscedasticity can lead to inefficient estimates of the regression coefficients. The standard errors of the coefficients can be biased, leading to unreliable hypothesis tests and confidence intervals.

    Inference Validity: Many statistical tests and confidence intervals rely on the assumption of homoscedasticity. When heteroscedasticity is present, these inferences can be incorrect.

    Model Diagnostics: Detecting heteroscedasticity is an important step in model diagnostics, as it indicates that the model may need to be improved or transformed to address the issue.

# Generate sample data with heteroscedasticity
np.random.seed(0)
X = np.linspace(0, 100, 500)
y = 2 * X + np.random.normal(0, 1 + 0.1 * X, size=X.shape)  # Increasing variance

# Fit a linear regression model
X_reshaped = X.reshape(-1, 1)
model = LinearRegression()
model.fit(X_reshaped, y)
y_pred = model.predict(X_reshaped)

# Calculate residuals
residuals = y - y_pred

# Create a DataFrame for plotting
df = pd.DataFrame({'X': X, 'y': y, 'y_pred': y_pred, 'residuals': residuals})

# Plot the residuals
plt.figure(figsize=(12, 6))
sns.scatterplot(x='X', y='residuals', data=df)
plt.axhline(0, color='red', linestyle='--')
plt.title('Residual Plot (Heteroscedasticity)')
plt.xlabel('Fitted Values (X)')
plt.ylabel('Residuals')
plt.show()

Interpretation of the Heteroscedastic Residual Plot:

    Funnel Shape: The residuals exhibit a funnel shape, widening as the fitted values increase. This indicates increasing variance, a hallmark of heteroscedasticity.

    Pattern: The systematic increase in residual spread suggests that the assumption of homoscedasticity is violated.

Summary:

    Homoscedasticity: Constant variance of residuals; ideal for linear regression.

    Heteroscedasticity: Changing variance of residuals; can bias model estimates.

    Residual Plot: A key diagnostic tool for checking homoscedasticity.
