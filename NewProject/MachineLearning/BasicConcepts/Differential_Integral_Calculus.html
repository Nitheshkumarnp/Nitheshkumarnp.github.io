Differential and Integral Calculus:

Differential Calculus:

Definition: Differential calculus is a branch of mathematics that deals with the study of how functions change when their inputs change. It focuses on the concept of the derivative, which measures the rate of change of a function with respect to one of its variables. Key Concepts:

    Function: A relation between a set of inputs and a set of possible outputs where each input is related to exactly one output.

    Derivative: The derivative of a function () at a point x is a measure of how () changes as  changes. It is denoted as () or /.

    Slope of a Tangent Line: The derivative at a point  gives the slope of the tangent line to the curve at that point.

    Rate of Change: The derivative represents the rate of change of the function's output with respect to the input.

Integral Calculus:

Definition:

    Integral calculus is the branch of mathematics concerned with the concept of integration, which is essentially the reverse process of differentiation. It deals with finding the total accumulation of quantities and the areas under and between curves.

Key Concepts:

    Integral: The integral of a function represents the accumulation of quantities, such as areas under a curve. It is denoted by the integral sign .

    Indefinite Integral: An integral without specified limits, representing a family of functions. It includes a constant of integration, , because the differentiation of a constant is zero.

    Definite Integral: An integral with specified limits, representing a specific numerical value. It calculates the area under the curve between two points.

    Fundamental Theorem of Calculus: This theorem links differentiation and integration, stating that differentiation and integration are inverse processes. Specifically, if () is an antiderivative of (), then: ()=()()

import numpy as np
import matplotlib.pyplot as plt

# Define the range and step size
x_values = np.arange(1, 5.1, 0.1)

# Define the functions
f_x = x_values**3
f_prime_x = 3 * x_values**2
cumulative_f_prime_x = np.cumsum(f_prime_x) * 0.1  # Multiply by 0.1 to account for the step size

# Create the plot
plt.figure(figsize=(10, 6))

# Plot f(x) = x^3
plt.plot(x_values, f_x, label=r'$f(x) = x^3$', color='b', marker='o')

# Plot f'(x) = 3x^2
plt.plot(x_values, f_prime_x, label=r"$f'(x) = 3x^2$", color='r', marker='x')

# Plot cumulative f'(x)
plt.plot(x_values, cumulative_f_prime_x, label="Cumulative $f'(x)$", color='g', marker='s')

# Add titles and labels
plt.title("Visualization of $f(x) = x^3$, $f'(x) = 3x^2$, and Cumulative $f'(x)$")
plt.xlabel('x')
plt.ylabel('Value')
plt.legend()
plt.grid(True)

# Show the plot
plt.show()

    Scenario: Suppose we have a simple linear regression problem where we want to fit a line to a set of data points.

            Cost Function: The cost function J() measures how well the model's predictions match the actual data. For linear regression, it is often the mean squared error.

import numpy as np
import matplotlib.pyplot as plt

# Generate synthetic data
np.random.seed(0)
X = 2 * np.random.rand(100, 1)
y = 4 + 3 * X + np.random.randn(100, 1)

# Initialize parameters
theta = np.random.randn(2, 1)  # Random initialization
X_b = np.c_[np.ones((100, 1)), X]  # Add x0 = 1 to each instance

# Learning rate and iterations
learning_rate = 0.1
n_iterations = 1000
m = 100

# Gradient Descent
for iteration in range(n_iterations):
    gradients = 2/m * X_b.T.dot(X_b.dot(theta) - y)
    theta = theta - learning_rate * gradients

# Final parameters
print("Theta:", theta)

# Plot the data and the regression line
plt.plot(X, y, "b.")
plt.plot(X, X_b.dot(theta), "r-", label="Predictions")
plt.xlabel("$x_1$")
plt.ylabel("$y$")
plt.legend()
plt.title("Linear Regression with Gradient Descent")
plt.show()

Summary:

Differential Calculus:

    Focuses on finding the rate of change (derivatives) of functions. It answers questions like "How fast is a quantity changing at a given instant?"

Integral Calculus:

    Focuses on finding the total accumulation (integrals) of quantities. It answers questions like "What is the total quantity accumulated over an interval?"
