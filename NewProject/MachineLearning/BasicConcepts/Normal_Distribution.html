Normal Distribution

Normal Distribution vs Non Normal Distribution

Normal Distribution:

Characteristics:

    Symmetrical bell-shaped curve.
    Mean, median, and mode are equal.
    Defined by two parameters: mean () and standard deviation ().

Advantages:

    Many statistical tests (e.g., t-tests, ANOVA, linear regression) assume normality, making analysis straightforward.
    Outcomes tend to cluster around the mean, making predictions more reliable.
    Standard metrics like mean and standard deviation are meaningful and easily interpretable.
    Enables inference from sample statistics to population parameters when sample sizes are large.

Non Normal Distribution:

Characteristics:

    Can be skewed (left or right), have heavy tails, or be multimodal.
    Mean, median, and mode may differ.
    Described by more complex parameters, depending on the distribution type.

Challenges:

    Many parametric tests assume normality, so non-normal data may violate these assumptions, leading to incorrect conclusions.
    Descriptive statistics like mean and standard deviation might not fully describe the data's behavior.
    Predictions can be less reliable, especially if the distribution is heavily skewed or has outliers.

To find Normal Distribution, there are various ways,

    Visualization,
    Statistical methods,
    Skewness and Kurtosis

Visualization

Histogram:

    A histogram provides a visual representation of the distribution of the data.

Q-Q Plot (Quantile-Quantile Plot):

    A Q-Q plot compares the quantiles of the dataset against the quantiles of a normal distribution. If the data is normally distributed, the points will lie approximately along the line y=x.
In[2]:
import numpy as np
import matplotlib.pyplot as plt
import scipy.stats as stats
from scipy.stats import shapiro, kstest, skew, kurtosis

# Generate some data
data = np.random.normal(loc=0, scale=1, size=1000)  # Normally distributed data

# Plot histogram
plt.figure(figsize=(10, 6))
plt.hist(data, bins=30, edgecolor='k', alpha=0.7)
plt.xlabel('Value')
plt.ylabel('Frequency')
plt.title('Histogram')
plt.show()

# Generate Q-Q plot
plt.figure(figsize=(10, 6))
stats.probplot(data, dist="norm", plot=plt)
plt.title('Q-Q Plot')
plt.show()

Statistical methods

Shapiro-Wilk Test:

    The Shapiro-Wilk test tests the null hypothesis that the data was drawn from a normal distribution. A low p-value (typically  0.05) indicates that the null hypothesis can be rejected, meaning the data is not normally distributed.

Kolmogorov-Smirnov Test:

    The Kolmogorov-Smirnov test compares the sample with a reference probability distribution (normal distribution in this case).
In[3]:
# Shapiro-Wilk test
stat, p = shapiro(data)
print('Shapiro-Wilk Test: Statistics=%.3f, p=%.3f' % (stat, p))
alpha = 0.05
if p > alpha:
    print('Sample looks Gaussian (fail to reject H0)')
else:
    print('Sample does not look Gaussian (reject H0)')

# Kolmogorov-Smirnov test
stat, p = kstest(data, 'norm')
print('Kolmogorov-Smirnov Test: Statistics=%.3f, p=%.3f' % (stat, p))
if p > alpha:
    print('Sample looks Gaussian (fail to reject H0)')
else:
    print('Sample does not look Gaussian (reject H0)')

Skewness and Kurtosis

Skewness:

    Skewness measures the asymmetry of the data distribution. For a normal distribution, skewness is 0.

Kurtosis:

    Kurtosis measures the "tailedness" of the data distribution. For a normal distribution, kurtosis is 3.
In[4]:
# Calculate skewness
skewness = skew(data)
print('Skewness: %.3f' % skewness)

# Calculate kurtosis
kurt = kurtosis(data, fisher=False)  # Use fisher=False to get kurtosis relative to a normal distribution
print('Kurtosis: %.3f' % kurt)

Skewness: 0.101
Kurtosis: 2.975

What if Data is not normally distributed

There are several ways to handle non normal distributed data:

    Transformation,
    Non-parametric Methods: Use statistical methods that do not assume a normal distribution,
    Robust Statistical Methods: Use methods that are robust to deviations from normality.
In[5]:
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import boxcox, yeojohnson, normaltest

# Generate non-normally distributed data
data = np.random.exponential(scale=2, size=1000)

# Plot original data
plt.figure(figsize=(10, 6))
plt.hist(data, bins=30, edgecolor='k', alpha=0.7)
plt.title('Original Data (Exponential Distribution)')
plt.show()

# Apply log transformation
data_log = np.log(data)

# Apply square root transformation
data_sqrt = np.sqrt(data)

# Apply Box-Cox transformation
data_boxcox, _ = boxcox(data + 1)  # Box-Cox requires positive data

# Apply Yeo-Johnson transformation
data_yeojohnson, _ = yeojohnson(data)

# Plot transformed data
transformations = [data_log, data_sqrt, data_boxcox, data_yeojohnson]
titles = ['Log Transformation', 'Square Root Transformation', 'Box-Cox Transformation', 'Yeo-Johnson Transformation']

plt.figure(figsize=(15, 10))
for i, (transformed_data, title) in enumerate(zip(transformations, titles), 1):
    plt.subplot(2, 2, i)
    plt.hist(transformed_data, bins=30, edgecolor='k', alpha=0.7)
    plt.title(title)
plt.tight_layout()
plt.show()

Advantages of Being Normally Distributed:

    Statistical Assumptions: Many statistical tests (e.g., t-tests, ANOVA) assume normality in the data.
    Parameter Estimation: Parameters like mean and standard deviation are most informative and interpretable when the data is normally distributed.
    Predictive Modeling: Some machine learning algorithms (e.g., linear regression) perform better with normally distributed variables.
    Central Limit Theorem (CLT): According to CLT, the distribution of sample means approaches normality as sample size increases, regardless of the original distribution. This property simplifies hypothesis testing and confidence interval estimation.
    Data Transformation and Scaling: Normally distributed data often lead to better results when standardizing (z-score normalization).

Importance of Normal Distribution:

    Foundation of Many Techniques: Normal distribution is a foundational assumption for many statistical techniques and machine learning algorithms.
    Symmetry and Predictability: Normal distributions are symmetric, making them predictable and easier to work with.
    Optimal for Certain Metrics: Metrics such as mean and standard deviation are most meaningful when the data is normally distributed.
    Simplifies Analysis: Normality simplifies the mathematical treatment of the data, making analyses and interpretations more straightforward.
